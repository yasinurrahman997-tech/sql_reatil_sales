# SQL Retail Sales Analysis

This project analyzes retail sales data using SQL queries and a CSV dataset.  
It includes:
- SQL scripts for data cleaning and analysis  
- Retail sales dataset (`SQL - Retail Sales Analysis_utf.csv`)

## Files
- `SQL_retail_sales_project.sql` → Main SQL queries  
- `SQL - Retail Sales Analysis_utf.csv` → Dataset  
- `README.md` → Project documentation

## Objective
To extract insights such as top-performing products, total revenue trends, and customer purchasing patterns.
